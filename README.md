# Thinkora 🧠

An AI-powered chat application built with the MERN stack, featuring Google Authentication and personalized chat history for each user.

## Features
- 🔐 Google Authentication via Firebase
- 💬 Real-time AI responses powered by GPT-4o-mini
- 🗂️ Per-user chat history stored in MongoDB
- 🗑️ Create, switch, and delete chat threads
- 💻 Code syntax highlighting in responses
- 🌙 Clean dark UI

## Tech Stack
**Frontend:** React 19, Vite, Firebase Auth  
**Backend:** Node.js, Express.js  
**Database:** MongoDB Atlas  
**AI:** OpenRouter (GPT-4o-mini)  

## Setup

### Backend
```bash
cd Backend
npm install
npm run dev
```

### Frontend
```bash
cd Frontend
npm install
npm run dev
```
